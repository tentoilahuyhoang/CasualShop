# CasualShop-API

Public branch for deploy.

### the API

```
https://casualshop-api-production.up.railway.app/
```

### API Call
*Get all product*
```
/api/v1/product
```
*Get all category*
```
/api/v1/category
```
*Get all discount*
```
/api/v1/discount
```

This app is deployed on [railway](https://railway.app/)

### Enviroment Variables

| var name      | value    |
| --------------| -------- |
| PORT          | value    |
| JWT_SECRET    | value    |
| MYSQLHOST     | mysql    |
| MYSQLPORT     | 3306     |
| MYSQLUSER     | public   |
| MYSQLPASSWORD | password |
